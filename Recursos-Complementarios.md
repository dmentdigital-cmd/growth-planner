## Recursos Anthropic (Agentes con Claude)

### 1. Building Effective Agents
📚 **Referencia:** https://www.anthropic.com/engineering/building-effective-agents

**Por qué leerlo:** Guía oficial de Anthropic sobre diseño de agentes con Claude. Cubre arquitecturas, patrones y mejores prácticas para construir agentes robustos y efectivos.

**Relación con el curso:** Complementa la sección [[Flujos-Nodos-vs-Agentes-IA#Agentes con LLM (Autónomos)|Agentes con LLM]]

**Aplicable a Dmente Digital:** Especialmente para agentes de calificación de leads y customer success.

---

### 2. Multi-Agent Research System
📚 **Referencia:** https://www.anthropic.com/engineering/multi-agent-research-system

**Por qué leerlo:** Análisis de sistemas donde múltiples agentes colaboran para resolver problemas complejos. Muestra cómo orquestar agentes en paralelo y serie.

**Relación con el curso:** Avanzado, para [[Clasificacion-Iniciativas#Apuestas de largo plazo|apuestas de largo plazo]] que requieren múltiples agentes.

**Aplicable a Dmente Digital:** Para escenarios complejos como análisis de leads, enriquecimiento de datos y decisiones multi-criterio.

## APIs y SDKs para Agentes

### Claude API (Anthropic)
📚 **Documentación:** https://anthropic.com/api

**Por qué:** Acceso directo a Claude para construir agentes con herramientas, memoria y razonamiento.

**Aplicable:** Agentes custom, integración en sistemas propios.

---

### OpenAI API
📚 **Documentación:** https://platform.openai.com/docs/agents

**Por qué:** Alternativa con GPT-4, compatible con muchas herramientas.

**Aplicable:** Si necesitas coexistencia con ecosistema OpenAI.
