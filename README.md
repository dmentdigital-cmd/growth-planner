# 🚀 Growth Hacking Planner - Dmente Digital

**Herramienta interactiva de control de experimentos y planning de growth marketing**

Aplicación web diseñada para ejecutar el Growth Hacking Loop: Objetivos → Ideas → Priorización → Planificación de Tests → Ejecución → Análisis de Resultados.

## ✨ Características

✅ **6 Pasos del Growth Hacking Loop**
- Definir objetivos y métricas norte
- Lluvia de ideas (brainstorm)
- Priorización con matriz ICE (Impact, Confidence, Ease)
- Planificación de experimentos con hipótesis
- Ejecución y seguimiento de tests (Kanban)
- Análisis de resultados y toma de decisiones

✅ **Funcionalidades**
- Interfaz intuitiva con navegación circular
- Persistencia local con localStorage
- Exportar proyectos a JSON
- Branding Dmente Digital completo
- Totalmente responsive (mobile-first)
- Sin dependencias externas (Vanilla JS + Tailwind CSS)
