## Cómo Usar Este Curso

### Si eres nuevo en Growth + IA
1. Lee **Módulo 1** y **Módulo 2** para mentalidad y contexto
2. Visualiza la **Línea de Tiempo (Módulo 3)** para entender el viaje
3. Estudia **Módulo 4** para aprender el framework práctico
4. Ve a **Recursos Prácticos** para empezar
5. Consulta **Recursos Complementarios** para profundizar

### Si tienes experiencia en Growth
1. Salta a **Módulo 4** (Framework de 5 Pasos)
2. Usa **I.P.A.R.E.** como referencia rápida
3. Consulta **Clasificación de Iniciativas** para priorizar tu backlog
4. Ve a **Flujos vs Agentes** para elegir stack técnico
5. Copia plantillas de **Recursos Prácticos**

### Si necesitas validar una apuesta específica
1. Usa **Clasificación de Iniciativas** para categorizarla
2. Aplica **Framework de 5 Pasos** (o **I.P.A.R.E.**) para evaluarla
3. Consulta **Flujos vs Agentes** para arquitectura
4. Usa **Plantilla de Priorización** para documentarla
5. Lee **Recursos Complementarios** si necesitas profundizar
